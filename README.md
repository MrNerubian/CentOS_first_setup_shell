## 2、Useage：

- 备份功能为强制备份，不能关闭，采用时间戳命名，精确值秒，保障文件不会被覆盖

- 主机名设置需要重启终端才能生效

- 网络配置中，默认网关为x.x.x.2，有需要可手动再更改

- 如果采用远程终端连接，建议跳过重启网络功能，直接执行下一步重启服务器

- 配置本地光盘源需要连接光驱
